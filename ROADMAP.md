# Roadmap

## Version actual

- Web local-first.
- Electron escritorio.
- Importacion de PDF, TXT y Markdown.
- Filtros visuales.
- Ajustes de lectura.
- Modo paginado y modo flujo.
- Biblioteca local.
- Favoritos.
- Progreso de lectura.
- Responsive base.

## Proximas mejoras

- Mejorar responsive movil.
- Mas robustez del parser PDF.
- Modo PDF original.
- Notas y subrayado.
- Exportar notas.
- Soporte EPUB.
- OCR para PDFs escaneados.
- Lectura en voz alta.
- Instalador Windows.
- APK Android futura.
- Mejoras de accesibilidad.

## No implementado todavia

- OCR.
- EPUB.
- Sincronizacion en nube.
- APK.
- Instalador final.
- Cuentas de usuario.
- Reviews, ratings o funciones sociales.

# Papel Vivo v0.1.0 Beta

Primera beta preparada para publicar como codigo fuente en GitHub y distribuir artefactos desde GitHub Releases.

## Incluye

- Version web local-first para lectura de documentos sin subir archivos a un servidor.
- Instalador Windows.
- Version portable Windows.
- APK Android firmada localmente.
- Lector con modo paginado y modo flujo.
- Ajustes visuales de lectura.
- Biblioteca local.
- Favoritos.
- Punto de lectura guardado.
- Buscador interno.
- Notas por pagina.

## Avisos

- Windows puede mostrar SmartScreen porque el instalador no esta firmado digitalmente todavia.
- Android puede pedir permitir instalacion desde fuentes externas porque la APK no viene de Google Play.
- La app es una beta y puede contener fallos.
- La keystore Android no se publica ni se incluye en el repositorio.

## Instalacion

### Windows Setup

Descargar el instalador desde GitHub Releases y ejecutarlo normalmente.

### Windows Portable

Descargar la version portable desde GitHub Releases y ejecutarla sin instalacion.

### Android APK

Descargar la APK desde GitHub Releases, permitir instalacion desde fuentes externas si Android lo solicita e instalarla.
